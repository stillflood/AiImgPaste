# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.0] - 2025-11-13

### 新增
- 🌐 多语言支持（中文/English），可在托盘菜单切换
- ⚡ 异步图片保存模式，提升粘贴响应速度
- 🔧 异步保存开关，可在设置中配置同步/异步模式

### 改进
- 🎯 优化粘贴可靠性，使用 `SendInput "{Raw}"` 直接发送文本
- ⏱️ 增加剪贴板等待超时检测（ClipWait）
- 📝 强制显示语言切换通知，不受通知开关控制

### 修复
- 🐛 修复 PowerShell 终端粘贴失败问题（需要管理员权限）
- 🔄 修复 AutoHotkey v2 箭头函数语法错误

### 文档
- 📖 添加 Windows 终端粘贴限制说明
- ⚠️ 添加管理员权限运行说明
- 🔍 完善故障排除指南

## [1.0.0] - 2025-11-13

### 新增
- 🎉 首个正式版本发布
- 📸 支持剪贴板图片自动保存为 PNG 格式
- 📁 支持从资源管理器复制的图片文件直接获取路径
- 🔗 自动生成 `@路径` 格式，便于 AI 工具识别
- 🐧 支持 Windows 和 WSL 路径格式互转（`C:\...` ↔ `/mnt/c/...`）
- ⌨️ 可配置的快捷键绑定（默认 Alt+Shift+V）
- ⚙️ 完整的托盘菜单设置界面
- 🚀 开机自启功能
- 🔔 可选的托盘通知提示
- 📝 支持自定义路径前缀（如 `@`, `@file` 等）
- ✏️ 支持路径后缀自定义文本（可包含换行符）
- 📂 按月份自动组织截图文件（`YYYY-MM/`）
- 🕐 文件名采用时间戳命名（`yyyyMMdd-HHmmss.png`）
- 💾 配置文件自动持久化保存
- 🎨 使用 sflood.ico 作为程序图标
- 🏗️ 提供 32 位和 64 位编译版本

### 技术特性
- 基于 AutoHotkey v2 开发
- 使用 PowerShell 进行图片格式转换
- 完整的 GUI 设置界面
- INI 配置文件管理
- 无需安装 AutoHotkey 的独立可执行文件

### 兼容性
- ✅ Windows 10/11
- ✅ WSL/WSL2 环境
- ✅ Claude、Cursor、GitHub Copilot 等 AI 工具
- ✅ 32 位和 64 位 Windows 系统

### 使用须知
- ⚠️ **部分终端（如 PowerShell）需要以管理员权限运行程序**才能正常粘贴文本
- 📝 管理员权限是 Windows 系统对模拟键盘输入的安全限制
