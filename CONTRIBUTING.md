# 贡献指南

感谢你考虑为 AiImgPaste 项目做出贡献！

## 如何贡献

### 报告 Bug

如果你发现了 bug，请创建一个 issue 并包含以下信息：

1. **Bug 描述**: 清楚简洁地描述问题
2. **重现步骤**: 详细的重现步骤
3. **预期行为**: 应该发生什么
4. **实际行为**: 实际发生了什么
5. **环境信息**: 
   - Windows 版本
   - AutoHotkey 版本
   - 使用的 AI 工具
6. **截图**: 如果适用，添加截图来解释问题

### 功能建议

我们欢迎新功能建议！请创建一个 issue 并包含：

1. **功能描述**: 详细描述建议的功能
2. **使用场景**: 说明为什么需要这个功能
3. **实现思路**: 如果有的话，简述可能的实现方式

### 代码贡献

1. **Fork 项目**
2. **创建功能分支** (`git checkout -b feature/AmazingFeature`)
3. **提交更改** (`git commit -m 'Add some AmazingFeature'`)
4. **推送到分支** (`git push origin feature/AmazingFeature`)
5. **创建 Pull Request**

### 代码规范

- 使用清晰的变量和函数命名
- 添加必要的注释，特别是复杂的逻辑
- 保持代码风格与现有代码一致
- 测试你的更改

### 提交消息规范

使用清晰的提交消息：

```
类型: 简短描述

详细描述（如果需要）

- 新增功能使用 feat:
- 修复 bug 使用 fix:
- 文档更新使用 docs:
- 代码重构使用 refactor:
```

## 开发环境设置

1. 安装 [AutoHotkey v2](https://www.autohotkey.com/download/)
2. 克隆仓库: `git clone https://github.com/StillFlood/AiImgPaste.git`
3. 双击 `AiPasteImg.ahk` 开始测试
